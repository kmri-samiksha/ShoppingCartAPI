# ShoppingCartAPI

## Overview
This is a .NET API project that helps to calculate the total price of items added in the shopping cart based on Client type.

## Running the API

1. Open the solution in Visual Studio or VS Code.
2. Run the application and it can be tested using Swagger

## Steps for API Testing

1. Open the project in Visual Studio
2. Right click on the solution(root folder) and click on Run Tests
3. It will open a Test Explorer window where you can see the Test Results.

## Testing using Swagger

1. Run the application
2. Input the data 
3. See the Response.
