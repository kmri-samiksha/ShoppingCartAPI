# ShoppingCartAPI

## Overview
This .NET API calculates the total price of items in a shopping cart based on the client type (Individual or Professional).

## Running the API

1. Open the solution in Visual Studio or VS Code.
2. Run the application. The API can be tested via Swagger.

## Steps for API Testing using xUnit Project 

1. Open the project in Visual Studio
2. Right click on the solution(root folder) and click on Run Tests
3. It will open a Test Explorer window where you can see the Test Results.

## Testing using Swagger

1. Run the application. You will see two API endpoints: one for Individual clients and one for Professional clients.
2. Click Try it out to view the request body.
3. Enter the test data and click Execute.
4. Check the response to verify the results.
